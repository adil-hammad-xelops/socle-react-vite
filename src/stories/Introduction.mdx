## üìö Getting Started

### Installation

All components are available through the centralized UI exports:

```typescript
import { Button, Input, Card, Modal } from '../components/ui';
```

### Basic Usage

```typescript
import { Button } from '../components/ui';

function MyComponent() {
  return (
    <Button variant="primary" onClick={() => alert('Clicked!')}>
      Click Me
    </Button>
  );
}
```

### With TypeScript

All components are fully typed:

```typescript
import { Button, ButtonProps } from '../components/ui';

const MyButton: React.FC<ButtonProps> = (props) => {
  return <Button {...props} />;
};
```

### Architecture Pattern

When building new features, follow this pattern:

```typescript
// 1. Page (just routing)
export function MyPage() {
  return <MyContainer />;
}

// 2. Container (logic, state, API)
export function MyContainer() {
  const [data, setData] = useState([]);
  useEffect(() => fetchData().then(setData), []);
  return <MyView data={data} />;
}

// 3. View (pure UI)
export function MyView({ data }) {
  return <div>{data.map(item => <Card key={item.id} />)}</div>;
}
```

## üöÄ Quick Start

### Installation

\`\`\`bash
npm install @mui/material @mui/icons-material @emotion/react @emotion/styled react-router-dom
\`\`\`

### Basic Usage

```tsx
import { Button, Card, Input } from './components/ui';

function MyComponent() {
  return (
    <Card title="Hello World">
      <Input label="Name" placeholder="Enter name" />
      <Button variant="primary">Submit</Button>
    </Card>
  );
}
```

### With Theme Provider

```tsx
import { ThemeProvider, CssBaseline } from '@mui/material';
import { lightTheme } from './components/ui/theme';

function App() {
  return (
    <ThemeProvider theme={lightTheme}>
      <CssBaseline />
      <MyComponent />
    </ThemeProvider>
  );
}
```
